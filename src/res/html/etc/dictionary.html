Wörterbuch

Hier findest du eine Sammlung von Fachbegriffen rund um das Thema Modding.

Dieses Wörterbuch wurde von mir und meinem Kumpel erstellt, um Modding-Begeisterten zu helfen, den Überblick zu behalten.

Viel Spaß beim Entdecken der Begriffe :D

Suchbegriff eingeben:

Homebrew

Software, die von Nutzern entwickelt wird, um auf einer Konsole ausgeführt zu werden...

Jailbreak

Der Prozess, bei dem eine Konsole so modifiziert wird, dass sie Software ausführen kann, die nicht vom Hersteller autorisiert wurde...

Custom Firmware

Eine modifizierte Version der Original-Firmware einer Konsole, die zusätzliche Funktionen bietet...

Modchip

Ein physisches Gerät, das in eine Konsole eingebaut wird, um sie zu modifizieren...

Brick

Der Zustand einer Konsole, die durch Fehlermodifikationen oder fehlgeschlagene Firmware-Updates unbrauchbar geworden ist...
