CFW Pack

Hier wird gezeigt wie sie das NiklasCFW Pack updaten

Öffne zuerst Ultrahand

Wähle darauf NiklasCFW Downloader aus

Im Update checker wählst du nun die NiklasCFW version aus bis ein Hacken kommt

Wähle nun im NiklasCFW Downloader CFW_Pack aus

Nun wird die neuste CFW Version gedownloadet. Warte bis der Download fertig ist.

Boote nun in Hekate

Du solltest sehen, dass dort V1.1.1.1 (oder die version, die du installierst), steht


Firmware updaten

Hier wird gezeigt, wie sie die Firmware updaten

Zuerst öffnen sie Ultrahand und gehen auf den NiklasCFW Downloader


Nun wählen sie Firmware aus

Klicken sie auf Firmware 20.1.1 (oder die version, die sie sehen)

Nun gehen sie in Sphaira und öffnen Daybrake

In Daybrake wählen sie den Ordner der Firmware aus und folgen der Installation

Wählen sie darauf Preserve Settings und Install Fat32 + ExFat aus

Das System startet Neu und die neue Firmware sollte installiert sein


Sie können in den Systemeinstellungen unter Konsole überprüfen, ob ihre Firmware aktualisiert wurde
